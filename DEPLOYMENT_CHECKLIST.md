# Чеклист развертывания на guestreviews.ru

## Перед развертыванием

### DNS и домен
- [ ] Домен guestreviews.ru зарегистрирован
- [ ] DNS A-запись указывает на IP сервера
- [ ] DNS A-запись для www.guestreviews.ru настроена
- [ ] DNS изменения применились (проверить через `dig guestreviews.ru`)

### Сервер
- [ ] Сервер арендован (минимум 2GB RAM, 2 CPU)
- [ ] Ubuntu 20.04+ или Debian 11+ установлена
- [ ] SSH доступ настроен
- [ ] Firewall настроен (порты 22, 80, 443)

### Подготовка
- [ ] Docker установлен
- [ ] Docker Compose установлен
- [ ] Git установлен (опционально)
- [ ] Код загружен на сервер

## Настройка

### Переменные окружения (.env)
- [ ] Файл .env создан из .env.production
- [ ] SECRET_KEY сгенерирован (32+ символов)
- [ ] POSTGRES_PASSWORD установлен (сильный пароль)
- [ ] REDIS_PASSWORD установлен (сильный пароль)
- [ ] SMTP настройки заполнены (для email)
- [ ] CORS_ORIGINS содержит правильные домены
- [ ] DEBUG=false
- [ ] ENVIRONMENT=production

### SSL сертификат
- [ ] Email для Let's Encrypt указан в scripts/setup_ssl.sh
- [ ] Скрипт setup_ssl.sh выполнен успешно
- [ ] Сертификат получен в certbot/conf/live/guestreviews.ru/
- [ ] Сертификат валиден (проверить через браузер)

### Nginx
- [ ] Конфигурация nginx/guestreviews.ru.conf проверена
- [ ] Пути к SSL сертификатам правильные
- [ ] Proxy настройки корректны

## Полезные команды

```bash
# Проверка статуса
docker-compose -f docker-compose.prod.yml ps

# Логи
docker-compose -f docker-compose.prod.yml logs -f

# Перезапуск
docker-compose -f docker-compose.prod.yml restart

# Обновление
git pull
docker-compose -f docker-compose.prod.yml up -d --build

# Бэкап
docker-compose -f docker-compose.prod.yml exec postgres \
  pg_dump -U postgres hotel_reviews > backup_$(date +%Y%m%d).sql

# Проверка SSL
openssl s_client -connect guestreviews.ru:443 -servername guestreviews.ru
```
